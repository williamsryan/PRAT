# PRAT Portfolio Site Assets

This folder contains small, static assets for adding PRAT to Ryan's portfolio site.

Files:

- `AGENT_PROMPT.md`: paste this into the site-building agent.
- `project-brief.md`: content notes and polished copy.
- `mosquitto-tls-demo.json`: structured demo data from a successful local smoke run.
- `pipeline.svg`: portable visual asset for the page.

The full generated smoke-run report is currently available locally at:

- `/tmp/prat-smoke/report.html`
- `/tmp/prat-smoke/report.json`
- `/tmp/prat-smoke/FDG.dot`

Those full artifacts are generated outputs, so this folder keeps only compact, source-controlled assets that are easy to copy into a portfolio repo.
